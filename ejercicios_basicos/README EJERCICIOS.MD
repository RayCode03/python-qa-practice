📌 Descripción

Se desarrolló una función en Python capaz de validar contraseñas según reglas comunes de seguridad.

Este ejercicio simula una validación típica de formularios web, muy común en testing QA.

🎯 Reglas de validación

Una contraseña es válida si cumple TODAS:

Mínimo 8 caracteres
Al menos 1 número
Al menos 1 letra mayúscula
Al menos 1 letra minúscula

La función retorna:

True → contraseña válida
False → contraseña inválida
